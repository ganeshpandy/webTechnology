IDENTIFIER:

Any name that is given by a programmer is called an identifier. A programmer in java is responsible for the following names.

  1. Class Name
  2. Method Name
  3. Varible Name

Rules for an identifier name

  • The name should begin with an alphabet.
  • Number are allowed but not in the beginning.
  • Special symbols"$"&"_"are allowed but not recommend
  • Keywords cannot be used as identifier
  • Blank spaces cannot be present in an identifier
